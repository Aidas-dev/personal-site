# Hello World

Welcome to my first blog post. This site is more than just a portfolio; it's an experiment in self-hosting and modern web engineering.

## Why Talos Linux?

Talos is a modern Linux distribution built for Kubernetes. It's immutable, ephemeral, and managed entirely through an API. There's no SSH, no shell, and no unnecessary packages. This makes it incredibly secure and easy to manage at scale.

## The Tech Stack

- **Frontend**: TanStack Start with React 19
- **Infrastructure**: Talos Linux on Bare Metal
- **Orchestration**: Kubernetes
- **Deployment**: Cloudflare Workers (via TanStack Start)
